오늘 방문한 사이트와 방문 횟수, 체류 시간을 한눈에 확인합니다.

시간대별 히트맵

24시간 중 언제 가장 활발하게 브라우징했는지 시각화합니다.

🔍

오늘의 관심사 키워드

방문한 페이지 제목을 분석해 오늘 내가 가장 관심을 가진 주제를 추출합니다.

설치 방법
Chrome 개발자 모드로 2분 안에 설치할 수 있습니다
1

저장소 다운로드

GitHub에서 ZIP으로 다운로드하거나 git clone합니다.

git clone https://github.com/Hyunuk22/Test_for_the_Escape.git
2

Chrome 확장 프로그램 페이지 열기

Chrome 주소창에 아래를 입력하세요.

chrome://extensions
3

개발자 모드 켜기

오른쪽 상단의 개발자 모드 토글을 활성화합니다.

4

압축 해제된 확장 프로그램 로드

압축 해제된 항목 로드를 클릭하고 다운로드한 폴더 안의 extension/ 폴더를 선택합니다.

5

확장 프로그램 아이콘 클릭

Chrome 툴바의 퍼즐 아이콘 → 오늘의 활동 분석기를 핀하고 클릭하면 바로 시작됩니다.
